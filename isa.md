
Регистры: R0-R15 8 бит, PC 16 бит, F 8 бит \
Регистры R12:R13 используют push/pop как SP \
Шина данных: 8 бит \
Шина адресса: 16 бит


| Категория  | Инструкция | Формат         | Описание                  |
|------------|------------|----------------|---------------------------|
| Передача   | MOV        | rd, rs         | rd <- rs                  |
|            | MOV        | rd, imm8       | rd <- imm8                |
|            | MOV        | rd, [rs1, rs2] | rd <- MEM[rs1 << 8 + rs2] |
|            | MOV        | [rd1, rd2], rs | MEM[rd1 << 8 + rd2] <- rs |
|            | PUSH       | rd             | push rd                   |
|            | POP        | rd             | rd <- pop                 |
|            | MOV        | rd, [addr16]   | rd <- MEM[addr16]         | ??????????
|            | MOV        | [addr16], rs   | MEM[addr16] <- rs         | ??????????
| Арифметика | ADD        | rd, rs         | rd <- rd + rs             |
|            | ADC        | rd, rs         | rd <- rd + rs + C         |
|            | SUB        | rd, rs         | rd <- rd - rs             |
|            | SBC        | rd, rs         | rd <- rd - rs - C         |
|            | INC        | rd             | rd <- rd + 1              |
|            | DEC        | rd             | rd <- rd - 1              |
|            | NEG        | rd             | rd <- -rd                 |
| Сдвиги     | SHL        | rd             | rd <- rd << 1             |
|            | SHR        | rd             | rd <- rd >> 1             |
| Логика     | AND        | rd, rs         | rd <- rd & rs             |
|            | OR         | rd, rs         | rd <- rd \| rs            |
|            | XOR        | rd, rs         | rd <- rd ^ rs             |
|            | NOT        | rd             | rd <- ~rd                 |
| Сравнение  | CMP        | rd, rs         | FLAGS <- rd - rs          |
| Переходы   | JMP        | addr16         | PC <- addr16              |
|            | JR         | [rs1, rs2]     | PC <- rs1 << 8 + rs2      |
|            | JZ         | addr16         | if Z=1;  PC <- addr16     |
|            | JNZ        | addr16         | if Z=0;  PC <- addr16     |
|            | JC         | addr16         | if C=1;  PC <- addr16     |
|            | JNC        | addr16         | if C=0;  PC <- addr16     |
|            | CALL       | addr16         | push PC, PC <- addr16     |
|            | RET        | —              | PC <- pop                 |
| Система    | NOP        | —              | Нет операции              |
|            | HALT       | —              | Остановка CPU             |

Формат команды: \
([инструкция 6 бит][00]) \
([инструкция 6 бит][01]) ([rd 4 бит][ rs 4 бит]) \
([инструкция 6 бит][10]) ([rd 4 бит][ XX 4 бит]) (imm 8 bit) \
([инструкция 6 бит][10]) ([rd 4 бит][rs1 4 бит]) ([rs2 4 бит][xx 4 бит]) \
([инструкция 6 бит][10]) (imm 8 bit            ) (imm 8 bit)s
