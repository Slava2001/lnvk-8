Регистры: R0-R15 8 бит, PC 16 бит, F 8 бит \
Регистры R12:R13 используют push/pop как SP \
Шина данных: 8 бит \
Шина адресса: 16 бит

Регистр флагов:
0: С - перенос (переполнение)
    ADD/ADC → перенос из старшего бита
    SUB/SBC/CMP → C=1 если rd ≥ rs, иначе 0
    SHL → вылетевший старший бит
    SHR → вылетевший младший бит
    AND/OR/XOR → всегда 0
1: Z - нулевой результат (result == 0)
2: S - флаг знака (отрицательный результат) (result[7])

|            | MOV        | rd, imm8       | rd <- imm8                |

|            | ADC        | rd, rs         | rd <- rd + rs + C         |
|            | SUB        | rd, rs         | rd <- rd - rs             |
|            | SBC        | rd, rs         | rd <- rd - rs - C         |
|            | INC        | rd             | rd <- rd + 1              |
|            | DEC        | rd             | rd <- rd - 1              |

|            | NOT        | rd             | rd <- ~rd                 |
| Сравнение  | CMP        | rd, rs         | FLAGS <- rd - rs          |
| Переходы   | JMP        | addr16         | PC <- addr16              |
|            | JR         | [rs1, rs2]     | PC <- rs1 << 8 + rs2      |
|            | JZ         | addr16         | if Z=1;  PC <- addr16     |
|            | JNZ        | addr16         | if Z=0;  PC <- addr16     |
|            | JC         | addr16         | if C=1;  PC <- addr16     |
|            | JNC        | addr16         | if C=0;  PC <- addr16     |

|            | RET        | —              | PC <- pop                 |
| Система    | NOP        | —              | Нет операции              |
|            | HALT       | —              | Остановка CPU             |

Формат команды: \
([инструкция 6 бит][00]) \
([инструкция 6 бит][01]) ([rd 4 бит][ rs 4 бит]) \
([инструкция 6 бит][01]) ([rd 4 бит][ XX 4 бит]) \
([инструкция 6 бит][10]) ([rd 4 бит][ XX 4 бит]) (imm 8 bit) \
([инструкция 6 бит][10]) (imm 8 bit            ) (imm 8 bit) \
([инструкция 6 бит][10]) ([rd 4 бит][rs1 4 бит]) ([rs2 4 бит][xx 4 бит]) \
([инструкция 6 бит][11]) ([rd 4 бит][ XX 4 бит]) (imm 8 bit            ) (imm 8 bit)
